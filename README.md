## 📌 프로젝트 소개
이 프로젝트는 **YOLOv8 모델을 사용하여 객체 탐지**를 수행하는 웹 애플리케이션입니다.  
Flask를 기반으로 동작하며, 사용자가 이미지를 업로드하면 AI가 실시간으로 객체를 탐지하고 결과를 표시합니다.
